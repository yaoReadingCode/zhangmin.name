【原题】复旦一研究生公布中国食品安全报告志愿者零成本工作 统计数据触目惊心
<div>实习生 陈竹 本报记者 周凯 《 中国青年报 》（ 2011年06月20日 &nbsp; 03 版）</div><div>&nbsp;</div><div>&nbsp; &nbsp; 人造的鸡蛋，塑料造的银鱼，糖水变的蜂蜜，猪肉变的牛肉&hellip;&hellip;在食品安全事件层出不穷的当下，人们对有毒食品的报道已见怪不怪。然而，复旦大学硕士三年级研究生吴恒却看不下去了，他用一个月时间，和34名志同道合者一起，做了一个《中国食品安全问题新闻资料库》。6月17日，相关报告发布并在网站上线的当天，吴恒接受了中国青年报记者的专访。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 一个月完成的报告</div><div>&nbsp;</div><div>&nbsp; &nbsp; 2011年5月11日，吴恒在人人网和博客上发布了一篇日志，召集志同道合的人来共同建立一个中国食品安全数据库。他在日志中说：&ldquo;我不是一个喜欢麻烦别人的人，特别是这种没有物质回报、付出了不一定得到别人理解的事情。但这一次，要我一个人来做完，的确有点难。&rdquo;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 让吴恒没有想到的是，1天之内，他就征集到了33位志愿者，大多来自他的近2000名人人网好友，包括22位&ldquo;80后&rdquo;，3位&ldquo;90后&rdquo;，甚至还有一名高三的学生。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 5月13日，团队正式开始运作。17天内，共查阅相关报道17268篇，约1000万字，从中筛选出有明确来源、有受害者的2107篇报道，制作了2849条记录，并为每篇报道提取了包括事发地、涉及食品的种类、对人体有害的原因等在内的关键词。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 5月27日，他们开始进行数据成果的标准化处理，参考了《食品安全法》，加之团队中有学食品专业的同学，最终将有害食品分为&ldquo;造假、过期、添加剂、混有异物、包装材料有问题、无证经营、产品不合格、检疫不合格、卫生不达标、其他&rdquo;10种。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 6月17日零点，发布资料库及调查报告的官方网站&ldquo;掷出窗外&rdquo;正式上线，《中国食品安全问题新闻资料库（2004-2011）》出炉，还提供地区、食品名、关键词等查询。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 这个项目的人力成本为零，所有搜集数据和建设网站的工作都是志愿者义务完成的，所有的开销来自购买域名和空间的200余元。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒的本科专业是空间信息科学，硕士专业则是历史地理学，他笑称，自己是&ldquo;有人文关怀的工科生&rdquo;，也是&ldquo;有工科背景的文科生&rdquo;。虽然两门专业都是&ldquo;打酱油&rdquo;路过，但这个报告的完成就得益于他受到的两方面训练。作为历史学科的学生，他对海量数据的收集、提取和整理相当在行；利用空间信息科学，他用自己熟悉的MapInfo软件制作出了各个年份的&ldquo;中国食品安全问题形势图&rdquo;以及时间、来源分布图，用Excel数据透视表制作了总曝光率、各地曝光率的图表统计。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 项目结束后，吴恒总结出，中国的食品安全问题有几个特点：涉及面特别广、违法手段特别狡猾、危害特别巨大、查处特别困难。 在&ldquo;掷出窗外&rdquo; 网站上，吴恒又专门添加了一项&ldquo;我要补充&rdquo;，以期待更多的网友在读到曝光食品安全危机的新闻时，将标题与链接添加进资料库，使资料库不断壮大和完善。他说，这项工作还未结束。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 让不明真相的群众认识有毒食品</div><div>&nbsp;</div><div>&nbsp; &nbsp; 今年4月，吴恒看到多地曝出用牛肉膏制作假牛肉的报道，这成为他做这个项目的动力所在。研二快结束时，他在寝室忙着写论文，习惯了点外卖，最爱吃的是学校后门小餐馆的铁板牛肉盖浇饭。他觉得性价比很高，一大盘牛肉才不到10元。当时，一个室友常和他一起点外卖，但从不点牛肉，说牛肉是假的。但吴恒不相信有假牛肉存在，他觉得口感和味道都不差。此前，他从来没有想到有毒食品距离自己这么近，牛肉膏的报道，使他突然发现自己成了受害者。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒由此开始关注食品安全事件。通过网络搜索，他找到一个叫做&ldquo;安全快报&rdquo;的资料库，该资料库收集了1万多条与食品相关的新闻报道，其中约 6000条与有毒食品相关。在看了这些报道后，他才发现，食品安全事件爆发的地点和频率是惊人的。当天中午，他纠结了很久，恶心得没有去吃午饭。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒自认是重度网络依赖者，每天上网超过8小时，关心时事，但这方面的大部分新闻他都没有读过，其他人就更不会知道了。于是，他开始想做一个关于中国近10年来食品安全状况的调查，让&ldquo;不明真相的群众&rdquo;重新认识自己身处的环境，并提醒大家，有毒食品一点都不遥远。&ldquo;知道真相但不告诉不明真相的人，这种不作为，在我看来是不可接受的。&rdquo;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 在召集日志中，吴恒引用了一段话，来自约翰&middot;多恩的《丧钟为谁而鸣》：&ldquo;没有人是一座孤岛，可以自全。每个人都是大陆的一片，整体的一部分。如果海水冲掉一块，欧洲就减小，如同一个海岬失掉一角，如同你的朋友或者你自己的领地失掉一块。任何人的死亡都是我的损失，因为我是人类的一员，因此，不要问丧钟为谁而鸣，它就为你而鸣。&rdquo;&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒说，永远不要对他人的苦难无动于衷，因为谁也无法保证下一个不是你。</div><div>&nbsp;</div><div>&nbsp; &nbsp; &ldquo;易粪相食&rdquo;与&ldquo;掷出窗外&rdquo;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 资料库完成后，吴恒又撰写了两篇报告：《易粪相食：中国食品安全状况调查（2004-2011）》与《掷出窗外：面对食品安全危机，我们应有的态度》。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 采用&ldquo;易粪相食&rdquo;这个词语，是受《左传》中的词语&ldquo;易子相食&rdquo;（意为灾荒年间，父母不忍心吃自己的子女，就互相交换子女来吃）的启发。五六年前，他曾在一个论坛上看到有网友以此发挥，用&ldquo;易粪相食&rdquo;来形容食品安全问题。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 今年4月，吴恒读到了上海查封染色馒头的报道。问题馒头的制作者面对记者采访时说：&ldquo;打死我都不会吃，饿死我都不会吃。&rdquo;这让他觉得又可气又可笑：如果每个有毒食品的制造者都选择这种思维方式，一个行业的人不吃自己生产的东西，觉得这样就安全了，这场博弈就没有赢家，大家都在&ldquo;易粪相食&rdquo;。在项目进行中，他又先后阅读到东莞和重庆两地用化粪池水熬地沟油的报道，这不就是现实生活中的&ldquo;易粪相食&rdquo;吗？&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; &ldquo;掷出窗外&rdquo;，则源于大洋彼岸另一个国家的食品安全史。当吴恒读到1906年，作家厄普顿&middot;辛克莱根据其在芝加哥一家肉食加工厂的生活体验写成的纪实小说《丛林》时，他发现，即使在美国，一个世纪前，其食品生产行业同样处于&ldquo;丛林状态&rdquo;。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 美国第26任总统西奥多&middot;罗斯福也在吃早餐时读到该书，突然&ldquo;大叫
一声，

跳起来，把口中尚未嚼完的食物吐出来，又把盘中剩下的一截香肠用力抛出窗外&rdquo;。罗斯福随后与辛克莱见面，推动通过了《纯净食品与药品法》，并创建了美国食品药品监督管理局的前身。美国食品药品监督管理局的创立，是世界食品安全史上有里程碑意义的事件，被称为&ldquo;美国食品安全守护神&rdquo;。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 选择这个名词，还有另一个私人的原因。2000年，吴恒读高中时，听学长说过一件事情：几年前，学校食堂曾将前一天卖不掉的饭菜第二天再卖，学生们忍无可忍，其中的活跃分子串联了各个班级，终于有一天，大家去食堂购买了早餐后，把油条、大饼全部扔在了教学楼前的空地上，随后，伙食的确有所改善。</div><div>&nbsp;</div><div>&nbsp; &nbsp; 在吴恒看来，在县城的高中里，学长们把油条从教学楼上扔下，与罗斯福在白宫把香肠从窗户扔出，相隔百年，相距万里，却是异曲同工：他们把早餐&ldquo;掷出窗外&rdquo;，他们在表达不满。&ldquo;为什么要这样做？我想那应该是因为他们相信，&lsquo;扫帚不到，灰尘照例不会自己跑掉&rsquo;，事情不会自己慢慢变好，需要外界的动力与刺激。&rdquo;他希望，&ldquo;掷出窗外&rdquo;是每个人面对食品安全问题应有的态度。</div><div>&nbsp;</div><div>&nbsp; &nbsp; &ldquo;这个国家就是因为这样一步步微小的改变而变得美好，希望我们做的事能引起更多人的关注。坚持自己的善良就是成功。&rdquo;项目的参与者赵悦说。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 评选中的&ldquo;达尔文奖&rdquo;&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 在网站上，吴恒还设立了一个专题，评选&ldquo;中国食品达尔文奖&rdquo;。他给出了10个候选名单，发动网友为最恶心的食品安全事件投票，奖金为人民币 1.4元，将颁发给首家曝光该类食品的新闻媒体。他笑称，这是取自&ldquo;一块死&rdquo;的谐音，如果食品安全危机继续下去，只能走向共同灭亡。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; &ldquo;达尔文奖&rdquo;得名于美国的温迪&middot;若斯科特，这位《纽约时报》畅销书作者不断搜集&ldquo;搬起石头砸自己的脚&rdquo;这类故事，其评选的&ldquo;达尔文奖&rdquo;旨在纪念这样一些&ldquo;笨死的人&rdquo;：&ldquo;他们通过愚蠢的方式毁灭了自我，大义凛然地把自己从人类的基因库中永远抹去，从而保证了人类长久的繁衍。&rdquo;&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒认为，自己评选出的有毒食品制造者，也正是那些试图以在食物中下毒这一愚蠢的方式毁灭自己所在民族，大义凛然地致力于把自己的民族从人类基因库中永远抹去，以保证剩下的人类长久繁衍的人。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 对话吴恒：对食品制造业应&ldquo;有罪推定&rdquo;&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 记者：项目运行中遇到过什么困难吗，与参与项目的其他志愿者合作有何感受？</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒：没有遇到过任何困难，一切都按照计划进行，除了花的时间太多，没空陪女朋友。本次调查，除我以外的34人中，只有5人与我在现实世界中有过面对面的交往，而且他们全部是通过网络获知这次调查的，可以说这是一个只会发生在互联网时代的故事。每个人的公益心和时间是有限的，但互联网的奇妙之处在于它可以把这些聚集起来，积少成多，完成一个人在现实世界中不能完成的事情，再反过来影响现实世界。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 网站的技术支持全部是由IT人士小白一个人完成的，他平日工作本已很辛苦，只能在下班后用自己的休息时间来一点一点写代码、建网站。此外，有的同学马上就要答辩了，有的同学自己的工作本身要加班，但也在百忙之中抽出时间来参与，还有同学电脑都快崩溃了，还坚持工作，大家付出的努力都让我很感动。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 记者：花时间做这个调查，希望达到什么目的？&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 吴恒：目的就是广而告之。因为只有更多的人开始关注这个问题了，群策群力，下情上达，事情才有改善的可能。也许这份调查报告一如此前的诸多新闻报道一样，似在湖心扔下一粒石子，激起了几片水花，但随后恢复平静，像什么也没发生。这种结局对我来说，并不会觉得太有所谓，我曾努力过，就已足够。很多年后回顾时，我可以坦然地告诉自己，当我们面对有毒食品的威胁时，我曾力所能及地和我的朋友们一块儿做过点什么。&nbsp;</div><div>&nbsp;</div><div>&nbsp; &nbsp; 我不能去告诉每个人，我也没有&ldquo;破开铁屋&rdquo;的能力，但至少我尝试着叫醒那些熟睡或装睡的人，让他们注意到这件事情的严肃性。因为我想，如果醒的人多了，也许他们能聪明地想出办法。最后的解决不可能是民间能够完成的，必须有政府的作为才能改变现状。我想引起周围的人注意，或者读此报告的读者中有一二人将来有机会成为政策制定者，能以&ldquo;身在公门好修行&rdquo;的心态来应对针对普通民众的食品安全危机，这是功德无量的善举。</div><div>&nbsp;</div><div>&nbsp;
